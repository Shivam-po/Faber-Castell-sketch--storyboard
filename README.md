# Faber-Castell-sketch--storyboard
Workflows to make a sketch into a live story.
workflow goes like:
sketch -> image/render
render -> story
story -> frames
frames -> video 
video -> full storyboard 


For these all the models and nodes used are open source.
1. Flux Kontext
2. Llama 3.1
3. Wan2.2
